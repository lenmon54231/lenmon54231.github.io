Array.apply(null, { length: 20 }) 和 new Array(20)

Array.apply(null, { length: 20 }) 生成的数组形式为[undifiend,undifiend,undifiend,…..],

new Array(20)等价于[，，，，]

后面使用.map方法的时候，map方法的限制为：map函数并不会遍历数组中没有初始化或者被delete的元素（有相同限制还有forEach, reduce方法）。

而new Array 并没有初始化。

所以，需要采用官方的写法。
