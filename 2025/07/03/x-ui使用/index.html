x-ui使用

X-UI使用记录（使用Ubuntu系统）

X-UI可以快速搭建面板

一键DD

  1. 下载脚本
  1. 安装系统(ubuntu或者debian)

OR

原作者git

搭建X-UI

  1. 执行脚本

  1. 输入面板登录账号密码和端口（8377）

  2. 打开8377端口

  1. 打开443端口（不知道是不是必须的，反正我执行了一次）
  1. 浏览器中打开64.176.37.182:8377（64.176.37.182替换成自己的服务器ip）

进行ssl证书申请

去cloudflare配置

cloudflare官网

  1. 添加自己的域名，然后cf将生成的dns解析地址，将地址填写到域名厂商dns解析处

  1. 先只使用cf的dns解析，暂时先不使用cdn，即：暂时先不开启橙色的云的图标

  1. cf 申请global API

右上角配置文件按钮 > API令牌 > API 密钥 > Global API Key 点击查看 以获取key

  1. X-UI执行申请ssl证书

需要提前准备以下数据：

1.cf注册邮箱

2.cf Global API Key

3.域名

  • 执行脚本

  • 选择16,按照提示输入3dweb.top域名和其他信息

  • 脚本最后会执行到成功

X-UI面板中添加入站地址

使用

在X-UI的入站列表里点击查看，即可出现弹窗，点击复制链接，然后复制到ray里即可

开启cdn

此时可以进入cf开启cdn，即将橙色的小云朵打开
