CentOS7安装Nginx

安装所需环境

Nginx 是 C语言 开发，建议在 Linux 上运行，当然，也可以安装 Windows 版本，本篇则使用 CentOS 7 作为安装环境。

顺序如下：

  1. gcc 安装

  2. PCRE pcre-devel 安装

  3. zlib 安装

  4. OpenSSL 安装

  5. 下载.tar.gz安装包

  6. 解压

  7. 配置

  8. 编译安装

  9. 启动、停止nginx

  10. 重启 nginx
